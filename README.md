# Neyo-The-Robot

**Neyo The Robot**

Neyo is a 90 cm humanoid robot built around a modular metal frame, servo-driven joints, and a clean, exposed-electronics aesthetic. Designed for real-world fabrication and experimentation, it blends mechanical simplicity with expandable control systems, making it an ideal platform for learning robotics, motion control, and embedded systems.

The robot features a fully articulated body with multiple degrees of freedom across the head, torso, arms, and legs, enabling lifelike movement and balance experimentation. Its structure is based on durable aluminum plates and standard servo brackets, ensuring that every component is practical to manufacture using common tools like CNC, laser cutting, or 3D printing.

At its core, Neyo integrates an ESP32 and Arduino-based control system, supported by PCA9685 servo drivers for precise motion control. Internal mounting spaces are designed for clean wiring, sensor integration, and efficient power management, including battery placement optimized for stability and center of gravity.

This project is not just about building a robot—it’s about understanding how mechanical design, electronics, and software come together to create motion. Neyo is meant to be modified, improved, and pushed further, whether that means implementing walking algorithms, adding vision systems, or refining balance and control.

**Key Features**

* ~90 cm humanoid robot with realistic proportions
* Modular aluminum frame with exposed mechanical design
* Servo-based joints with multiple degrees of freedom
* ESP32 + Arduino control architecture
* Expandable sensor integration (IMU, ultrasonic, camera)
* Designed for CAD modeling in Fusion 360 / SolidWorks / FreeCAD
* Clean internal layout for wiring and power systems

Neyo stands as a compact, buildable humanoid platform—simple in name, complex in potential.
